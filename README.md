# Proyecto final de BEI

Esta carpeta contiene los ejercicios para la evaluación final de la materia de Bioinformática y Estadística I. 

Contenido de las carpetas: 
- `data/`: Contiene los datos necesarios para realizar los ejercicios.
- `scripts/`: Contiene los scripts de R y Python para realizar los análisis.
- `results/`: Carpeta donde se deben guardar los resultados obtenidos de los análisis.
